Hello ! 
Welcome to my street fighter minigame.
This game is made for 2 players on one keyboard, one uses the aswd keys for movment and spacebar for punching,
and the other uses the arrow keys for movement and the arrow down key for punching.
winner is set by time/whenever the health drops to 0.
HAVE FUN :)

live demo : https://ripshtos.github.io/StreetFighter/
